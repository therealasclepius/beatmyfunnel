# Claude Code Project Config

## Working Style
- This is a business exploration project — expect strategy, research, and planning tasks alongside code.
- Use marketing skills (copywriting, page-cro, content-strategy, etc.) when relevant.
- Keep all business decisions and research organized in the docs/ folder.
- Keep code in src/ when development begins.

## Key Directories
- `docs/` — business plans, research, strategy docs
- `src/` — application code (when development begins)
- `assets/` — images, logos, brand assets
